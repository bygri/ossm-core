{% trans "OSSM is a re-envisioning of the multi-player strategic sports management game." %}

{% trans "Here's the first principles of the project." %}

  1. A focus on community, building links between people through both playing the game and developing it.
  2. Democratic design, where the direction of the game is determined by those who play it.
  3. Free to play and free to win, ensured by keeping the source code freely available for anybody to run their own game instance.
  4. There is no need for off-site management tools or paid add-ons, but API access is open for those who wish to use it.

{% trans "Some initial goals and ambitions:" %}

  1. Option for multiple linked teams (men's, women's, B-teams) in multiple sports (football, cricket). Football will be the first focus.
  2. Greater emphasis on tactics, with real-time substitutes and tactics changes available even during a match.
  3. Real-world weather and locations.
